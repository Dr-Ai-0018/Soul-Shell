# 用户画像模板

这个文件是仓库模板，可以提交。
实际使用时请复制为 `prompts/user_profile.md` 后按需填写，`user_profile.md` 已被 `.gitignore` 忽略，不应再提交。

## 核心人设

- 你平时怎么描述自己
- 你希望 Soul 记住的长期性格特点
- 你最讨厌的说话方式

## 技术背景

- 你常用的语言 / 框架 / 工具
- 你熟悉的系统环境
- 你现在主要在做什么类型的项目

## 工作习惯

- 你更喜欢简短直接，还是带解释的回答
- 你希望我默认先改代码，还是先讲思路
- 你对风险操作、确认步骤、日志输出有什么偏好

## 个人上下文

- 你最近在折腾的项目或目标
- 你常见的机器 / 服务器 / 部署环境
- 任何能帮助我更贴合你风格的背景信息

## 禁忌

- 不要提的内容
- 不要调侃的方向
- 不要暴露到仓库或公开环境的信息
